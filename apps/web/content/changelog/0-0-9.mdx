## Added

- Per-content-type `_redirects.json` files replace frontmatter `aliases` and `redirect_to` — declare slug migrations and retired documents in one place, then delete the source MDX
- Same-type (`toSlug`), cross-type (`toType` + `toSlug`), and anywhere (`toUrl`) redirect kinds with automatic translated-slug expansion from SQLite
- `localeRouting` config and `createUrlBuilder()` for safe, config-driven locale URL generation (`path-prefix` or `search-param`)

## Changed

- Redirects, alternates, sitemap, and runtime URL helpers now use the centralized URL builder instead of hardcoded locale prefix logic

## Removed

- Frontmatter `aliases` and `redirect_to` — `scribe validate` now errors if these fields are still present; use `_redirects.json` instead

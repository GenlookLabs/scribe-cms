## Added

- Templated asset fields: `field.asset()` declares a root-relative web path under your assets directory, with constraints, existence validation, and resolution to public URLs at read time
- Bodyless content types: a type can now omit the MDX body entirely, and translatability is derived from the schema instead of assumed
- Studio content browser: per-type document lists, global search across all content, and validation tooltips shown inline on documents
- Studio approximate MDX preview for document bodies
- A stale-while-revalidate cache for the studio's derived data (back references, asset graph, validation report), so pages stay fast while the project grows

## Changed

- `localeFallbacks` is now optional on the resolved config

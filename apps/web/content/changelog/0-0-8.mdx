## Added

- `listRoutableTypes()` on the Scribe client and project engine — returns content types that have a `path` template (the ones with public URLs, hreflang alternates, and sitemap entries)
- `isRoutableType` exported from `scribe-cms` and `scribe-cms/runtime` for apps and build scripts that need to filter routable types
